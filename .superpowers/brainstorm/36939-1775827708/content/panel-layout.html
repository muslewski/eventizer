Key Design Decisions

A

Collapsible sidebar on desktop

User can collapse the sidebar to icon-only mode (like VS Code). Good for users who want more content space.

B

Fixed sidebar on desktop

Always visible at 240px. Simpler implementation. Sidebar is not that wide, so content area is still spacious.
